# Domain — 로스트아크 도메인 지식

코드만 읽어서는 파악되지 않는 **게임 / 서비스 도메인** 맥락을 기록한다. API 필드의 의미, 게임 내 규칙, 캐릭터/원정대/각인 등 용어.

## 목록

- [streamer-research/README](./streamer-research/README.md) — 스트리머 데이터 수집 정책 (데이터 출처, 범위, 갱신 주기)
- [streamer-research/streamer-list](./streamer-research/streamer-list.md) — 대상 스트리머 목록
- 개인정보 관련 정책은 `security/` 에서 참조 (`streamer-research/privacy-notice.md`).

## 네이밍

- 로스트아크 원어(레이드/각인/카오스 게이트/원정대 등) 를 그대로 사용. 영어 직역 지양.
